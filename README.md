# Prácticas de Java ☕

Repositorio dedicado al aprendizaje y práctica del lenguaje Java, cubriendo desde conceptos básicos y algoritmos hasta Programación Orientada a Objetos (POO) e interfaces.

## 🛠️ Tecnologías y Requisitos
*   **JDK:** versión 17 o superior (o la que estés usando).
*   **IDE recomendado:** IntelliJ IDEA, Eclipse o VS Code.

## 📁 Estructura del Proyecto
*   **`java-condicionales-bucles-algoritmos/`**: Ejercicios prácticos usando estructuras de control (`if`, `switch`), bucles (`for`, `while`) y lógica algorítmica.
*   **`java-poo/`**: Proyectos enfocados en Programación Orientada a Objetos (Clases, Objetos, Herencia, Polimorfismo, Encapsulamiento).
*   **`java-interfaces/`**: Ejemplos del uso de interfaces y abstracción en Java.
*   **`java-pipeline/`**: Ejercicios/Pruebas con pipelines y flujos de datos.

## 🚀 Cómo ejecutar los ejercicios
1. Clona este repositorio:
   ```bash
   git clone <URL_DE_TU_REPOSITORIO>
   ```
2. Abre la carpeta del ejercicio que quieras revisar en tu IDE favorito.
3. Compila y ejecuta el archivo principal (usualmente conteniendo el método `main`).
